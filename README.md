# CRAVE 🍽️

A modern, minimalist recipe and meal-idea platform designed to help people discover what they can make and find recipes based on the ingredients they already have.

## 🌍 Languages

CRAVE is designed to support:

- 🇬🇧 ENG — English
- 🇱🇹 LT — Lithuanian

Users will be able to switch languages from the top of the website.

## ✨ Core Features

### 🔎 Search
Users can search through:

- Recipes
- Meal ideas
- Ingredients
- Tags
- Descriptions

### 🧺 Pantry Matching
Users can manually select ingredients they have from categories such as:

- Vegetables
- Fruits
- Meat
- Fish
- Dairy
- Grains
- Pantry basics
- Frozen foods

Recipes are ranked by how many ingredients the user already has.

Recipes with the highest ingredient match appear first.

Example:

- 100% match
- 80% match
- 60% match
- 40% match

A manual reset button allows users to clear their pantry selections.

### 🏷️ Manual Tags
The website owner manually creates and manages tags.

Example tags:

- Quick
- Healthy
- Cheap
- Vegetarian
- Vegan
- High Protein
- Breakfast
- Lunch
- Dinner
- Meal Prep

### 🍳 Recipes and Meal Ideas
CRAVE will support multiple types of content:

- Full recipes
- Meal ideas
- Recipe collections
- PNG recipe cards
- Canva-created recipe sheets

PNG recipe cards can be uploaded and later unpublished if necessary.

## 🔐 Content Management

Only the website owner can:

- Add recipes
- Edit recipes
- Remove recipes
- Add meal ideas
- Edit meal ideas
- Manage tags
- Upload PNG recipe cards
- Publish content
- Unpublish content

Content should be unpublished rather than permanently deleted whenever possible.

## 👤 Optional User Accounts

Users can browse and search without an account.

Accounts are optional and allow users to:

- Save recipes
- Favorite recipes
- Create collections
- Save shopping lists
- Create meal plans
- Save personal preferences
- Save dislikes if they choose to remember them
